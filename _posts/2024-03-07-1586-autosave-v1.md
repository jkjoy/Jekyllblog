## 代码
在 head 中插入
```html
<script src="https://cdnjs.sgcd.net/code-highlight/js/prism.js"></script>
<link rel="stylesheet" href="https://cdnjs.sgcd.net/code-highlight/css/prism.css"  />
```

## pjax回调
```php
$(document).on('pjax:end', function() {
  // 回调函数中调用 Prism 的高亮函数
  Prism.highlightAll();
});

```
