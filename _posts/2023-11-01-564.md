## 在根目录下创建swap文件夹
```
sudo mkdir /swap
cd  /swap
```
## 创建SWAPFILE
```
sudo fallocate -l 16G swapfile
```

## 查看刚才创建的
```
ls -lh /swapfile 
```

## 修改权限
```
sudo chmod 600 /swap/swapfile
```
## 将这个目录设置为交换分区
```
sudo mkswap /swap/swapfile
```

## 启用交换分区
```
sudo swapon /swap/swapfile
```
## 查看是否启用
```
sudo swapon --show
```
## 备份
```
sudo cp /etc/fstab /etc/fstab.bak
```
## 修改配置文件
```
echo '/swap/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
```
